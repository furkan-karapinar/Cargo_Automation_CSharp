# Kargo_Otomasyonu
Proje'nin demo dosyası yüklenmiştir. Projede kullanıcı, kargo gönderisi ve şube ekleyebilir, düzenleyebilir, silebilir, kargo durumunu görebilir, güncelleyebilir, desi ücreti hesaplayabilirsiniz. Admin girişi için kargo yönetimi seçeneği açılır. Personel girişinde bu seçenek yoktur.  Projeye ait bazı görüntüler aşağıda verilmiştir.
